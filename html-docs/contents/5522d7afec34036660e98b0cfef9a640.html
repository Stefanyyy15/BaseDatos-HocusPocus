Description

Ventas es una de las tablas principales ya que gestiona una gran cantidad de procesos, porque los empleados pueden hacer ventas, los clientes pueden comprar, las ventas se reflejan en las facturas y para realizar ventas se deben realizar transacciones. Por todas estas razones esta tabla sirve como puente para la mayoria de procesos.
