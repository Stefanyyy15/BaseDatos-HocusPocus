ERDEntity
Untitled :: Data Model1 :: PRODUCTOS

Description

La tabla productos se compone de los atributos basicos que comparten en comun todas las categorias de la tienda de dsifraces, nombre, descripcion, precio y un id_categoria que re afirma que muchos productos pueden tener una categoria. Esta tabla servirá tambien como tabla padre para las categorias predefinidas de la tienda.
